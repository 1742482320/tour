《Go 语言之旅》是官方 Go Tour 的中文翻译版。
要安装到本地，请先安装 Go，之后执行以下命令：

	$ go get -u github.com/Go-zh/tour/gotour
	$ cd $GOPATH/bin
	$ ./gotour

（如果安装过程中出现 `package` 或 `import`
字样的错误提示，那么说明依赖库的导入路径又挂了。
这时请猛戳 @OlingCat 并督促其解决= =||）
